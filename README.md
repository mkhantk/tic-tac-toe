# Tic Tac Toe

A simple Tic Tac Toe game built with **TypeScript** and a **custom game engine**, with the UI handled separately using React.

The project focuses on:
- Clear separation between game logic and UI
- Minimal global state
- Testable, UI-agnostic game rules

## Tech Stack
- TypeScript
- React (UI only)
- Vite
- Tailwind CSS

## Features
- Two-player gameplay
- Win and draw detection
- Score tracking
- Multiple rounds support

## Goal
This project was built to practice clean architecture, state flow, and separating core logic from presentation.
